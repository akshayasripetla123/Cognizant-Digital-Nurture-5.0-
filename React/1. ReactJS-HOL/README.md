# myfirstreact

This is a minimal Create React App project scaffold matching the "myfirstreact" hands-on lab.

## Setup

1. Make sure Node.js and npm are installed: https://nodejs.org/en/download/
2. Unzip this project.
3. Open a terminal in the `myfirstreact` folder and install dependencies:

   ```
   npm install
   ```

4. Start the app:

   ```
   npm start
   ```

5. Open your browser to `http://localhost:3000` — you should see the heading:

   **welcome to the first session of React**

## Notes

This project was generated as a lightweight version of `create-react-app` output
(no `node_modules` included, since those need to be installed with `npm install`
on your own machine). The `src/App.js` file already contains the required heading.
